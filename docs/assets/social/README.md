# Social media images

Generated by `npm run make-social` (`scripts/make-social.mjs`). Grab the one you need
by name:

| File | Size | Use |
| --- | --- | --- |
| `og.png` | 1200×630 | Link preview / Open Graph + Twitter card (websites, link unfurls) |
| `avatar.png` | 1024×1024 | Square profile picture (safe for circle crops) |
| `banner.png` | 1500×500 | X/Twitter profile header |

Edit the design or text in `scripts/make-social.mjs` and re-run `npm run make-social`.
